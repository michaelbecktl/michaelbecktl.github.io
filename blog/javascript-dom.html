Control Flow & Loops

When we talk about control flow, it is the way a script of code is being read. As we learned in HTML and CSS, codes are read from top to bottom and executed in order. With codes written at the bottom of a script possibly overwriting what's written beforehand. With Javascript, we can make use of conditionals and loops to change this control flow.

When we set a conditional is set, it allows a function or action written in the code script to either perform immediately or to wait for a condition to be fulfilled before executing. Where as for loops, they are functions that allow Javascript to execute a certain actions a specified amount of times in a compact line of codes. It helps so we don't have to copy and paste certain codes all over the script!
